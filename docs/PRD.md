| ------------ | ------------------------ |

| 使用终端     | 仅 PC 浏览器             |

## 3. 各环节详细说明

### 3.1 引导弹窗

| 项目                 | 说明                             |
| -------------------- | -------------------------------- |
| 触发时机             | 页面打开后自动弹出               |
| 是否需要勾选"已阅读" | **否**                           |
| 是否提示摄像头授权   | **否**（浏览器原生弹窗自行处理） |
| 关闭方式             | 用户点击「开始测评」按钮关闭     |

**弹窗内容（待修改）：**

> **欢迎使用心理测评系统**
>
> 测评流程说明：
> 1. 点击「开始测评」后，将进入 3 秒倒计时
> 2. 倒计时结束后，屏幕将展示一段文字材料，请您阅读
> 3. 录制过程中，摄像头将记录您的面部画面（共 40 秒）
> 4. 录制结束后系统自动提交，稍等片刻即可查看报告
>
> 请确保：面部处于画面中央、光线充足
>
> 【开始测评】

### 3.2 倒计时阶段

| 项目       | 说明                           |
| ---------- | ------------------------------ |
| 时长       | 3 秒                           |
| 展示内容   | 屏幕居中显示 3 → 2 → 1         |
| 摄像头预览 | 此时已开启，用户可看到自己画面 |
| 是否可取消 | 否                             |

### 3.3 录制阶段

| 项目             | 说明                                              |
| ---------------- | ------------------------------------------------- |
| 录制时长         | **40 秒**，固定不可中断（测试阶段可改为可中断的） |
| 视频格式         | **MP4**                                           |
| 屏幕布局         | 左侧/上方：文字材料；右侧/下方：摄像头预览画面    |
| 进度条           | 显示，标注剩余时间                                |
| 文字材料来源     | **前端写死**                                      |
| 录制结束行为     | 自动停止，自动上传                                |
| 用户能否中途取消 | **否**                                            |
